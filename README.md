📈 Stock Price Prediction System

A Machine Learning-based web application that predicts future stock prices using historical market data. The system analyzes past stock trends and applies regression models to forecast future price movements.

📌 Project Overview

The Stock Price Prediction System is designed to forecast stock closing prices using historical data.

It uses:

Data preprocessing techniques

Feature engineering

Machine Learning regression models

Interactive web interface (Streamlit / Flask)

The system helps users understand stock trends and visualize predictions effectively.
